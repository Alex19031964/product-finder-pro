# Product Finder Pro Web

A local browser-based product research tool for reseller opportunity analysis.

## Windows

1. Extract this ZIP.
2. Double-click `run_web_app_windows.bat`.
3. Your browser should open automatically.

If not, open:

http://localhost:8501

## What it includes

- Browser dashboard
- Supplier CSV import
- Marketplace signal CSV support
- ROI, landed cost, fee and profit calculations
- Low / medium / high opportunity scoring
- Revenue projection chart
- CSV and Excel export
- Local SQLite scan history
- API-ready placeholders for Keepa, Amazon SP-API and eBay

## Important

This app does not scrape websites. It is built for CSV imports and approved APIs / third-party providers.
